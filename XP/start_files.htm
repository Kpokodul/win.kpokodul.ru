Файлы и папки

Каждый рабочий элемент — или файл — может храниться в папке.

Интерфейс Windows XP упрощает сохранение файлов в наиболее подходящих местах. Сохраняйте текстовые файлы, изображения и музыкальные файлы в папках Мои документы, Мои рисунки и Моя музыка. Эти папки можно быстро открыть с помощью значков в правой части меню кнопки «Пуск», в них содержатся удобные ссылки для выполнения типичных задач.
