  Начальные сведения

Интерфейс Windows XP содержит новые экраны с понятным интерфейсом, упрощенные меню и многое другое. Откройте для себя новый модернизированный дизайн Windows XP. Ознакомьтесь с ключевыми задачами и основными командами.

Безопасность и простота работы

Windows XP превращает работу с персональным компьютером в удовольствие. Мощность, быстродействие, свежий новый вид, множество справочных данных, доступных при первой необходимости. Windows XP обладает всеми этими качествами, а также непревзойденной надежностью и безопасностью.

Дверь в мир цифрового аудио и видео

Откройте увлекательный мир цифрового аудио и видео дома, на работе и в Интернете. Наслаждайтесь цифровыми фотографиями, музыкой, видеозаписями, компьютерными играми.

Создание сетей дома и на работе

Организуйте совместный доступ к файлам, фотографиям, музыке, принтерам и даже подключениям к Интернету в сети с гарантированной конфиденциальностью и безопасностью.

Лучшее решение для бизнеса

Windows XP предоставляет проверенную надежность Microsoft Windows 2000, а также повышенные быстродействие и надежность.
